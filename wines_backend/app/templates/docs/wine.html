Wines API

Wine create

URL: {{ api_root_url }}/api/wine/

Method:   POST

Required Header:   Authorization 

Token f6c89fd05539c22c976792cd971ea78d67b430c9

Required Fields:   

image : file
image_description : string
title : string

Available Fields:   

news_link : string
text : string
tags_{number} : tag id

Translatable Fields: translations[lang][field] 

title
image_description
text : string

Example request Form-Data:

translations[en][title]
translations[ru][image_description]
image
partner_link
tags_1
